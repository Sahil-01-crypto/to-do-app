## 📌 About The Project

A clean and interactive **To-Do List Web App** built using **vanilla JavaScript**.
It helps users manage daily tasks efficiently with real-time updates and persistent storage.

---

## ✨ Features

✔️ Add tasks
✔️ Delete tasks
✔️ Edit tasks (double-click support)
✔️ Mark tasks as completed
✔️ Task counter (Total / Done / Pending)
✔️ Local Storage support (data saved after refresh)
